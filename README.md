# STK - Send to Kindle

STK is a Chrome extension that captures web page content, converts it to EPUB, and delivers it to Kindle by direct download or EmailJS-based send.

## Overall Status

- Project phase status: implemented through scaffold, extraction, EPUB generation, popup/options UI, background worker, tests, and packaging.
- Build output: `dist/` is generated by webpack and can be loaded as an unpacked extension.
- Package artifact: `stk-extension.zip` can be produced via `npm run pack`.

## Features

- Auto capture mode using `@mozilla/readability`
- Manual capture mode with hover highlight and click selection
- Metadata editing before EPUB generation (title, author, language)
- EPUB generation with HTML sanitization and image embedding
- Kindle device presets and typography customization
- Domain selector overrides for auto mode
- Delivery options:
  - Download EPUB locally
  - Send to Kindle email via EmailJS

## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Build extension assets:

```bash
npm run build
```

3. Load in Chrome:

- Open `chrome://extensions`
- Enable Developer mode
- Click Load unpacked
- Select `dist/`

4. Open the extension popup and capture a page.
